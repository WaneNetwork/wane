# Code of Conduct

## Our pledge

We want participation in the Wane project to be a productive and respectful
experience for everyone, regardless of background or level of experience. As
contributors and maintainers, we pledge to keep this an open and welcoming
community.

## Our standards

Behavior that helps the community:

- Using clear, considerate language
- Respecting differing viewpoints and technical opinions
- Accepting constructive feedback gracefully
- Focusing on what is best for the project and its users

Behavior that is not acceptable:

- Personal attacks, insults, or demeaning comments
- Public or private harassment of any kind
- Publishing others' private information without permission
- Sustained disruption of discussions, issues, or pull requests

## Scope

This Code of Conduct applies within all project spaces (the repository, issues,
pull requests, and discussions) and in public spaces when an individual is
representing the project.

## Enforcement

Maintainers are responsible for clarifying these standards and may remove,
edit, or reject comments, commits, code, issues, and other contributions that
do not align with this Code of Conduct.

Instances of unacceptable behavior may be reported to the maintainers at
conduct@wane.network. All reports will be reviewed and handled in a way that
respects the privacy and safety of the reporter.

## Attribution

This Code of Conduct is adapted from common open source community guidelines.
